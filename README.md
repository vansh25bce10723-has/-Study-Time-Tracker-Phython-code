Study Time Tracker

This is a small Python project where I made a simple Study Time Tracker. The main idea of this project is to keep a record of how long a student studies. Many times we think we study for long hours, but actually we donâ€™t know the exact time. So this program helps in tracking it.

The user can start a study session, stop it and then the program will calculate how much time was spent. All sessions are stored and the user can check the report any time. When the user exits the program, it also saves the report in a text file called study_report.txt.

This project is made using only basic Python concepts like loops, lists, conditions, file handling and the time module. It is simple to understand and useful for students to manage their time better.

Features

Start a study session

Stop the session and record the time

Show all study sessions

Calculate total study time

Save the final report in a text file

How to Run

Install Python on your system.

Download or copy the Python file study_tracker.py.

Open terminal or command prompt.

Run the program using:

python study_tracker.py


Follow the menu shown on the screen.

Output Example

Session Time: (some seconds)

Total Study Time: (sum of all sessions)
